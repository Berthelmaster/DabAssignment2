# DabAssignment2
Assignemnt2 Dab
Gruppe 9: Ali, Florent, Thomas & Zabih

I følgende aflevering er der blevet taget udgangspunkt i opgave 1 hvori gruppen skulle bruge tidligere blackboard database fra aflevering 1.
Dette skulle omskrives til "entity frameworks cores code-first technique"

Programmet fungerer ved at brugeren starter programmet uden debugging(Ctrl+f5), og dernææst bliver brugerens valgmuligheder udskrevet på skærmen, og brugeren kan vælge imellem 11 kommandoer.
Disse kommandoer bliver bestemt udfra brugerens input som er 1-11. Disse kommandoer bliver beskrevet i terminalen og instruktionerne står også beskrevet. Det skal bemærkes at databasen
allerede er seedet med "dummydata", som brugeren kan se via terminalen. Brugeren har også mulighed for at kunne tilføje/redigeren databasen. Programmet kører i en do-while(true) løkke,
og fortsætter derfor med at køre indtil brugeren manuelt lukker terminalen, men de ændringer der bliver fortaget bliver gemt i databasen. Det skal bemærkes at brugeren først skal 
oprette en lokal database ved navn "local" hvori alt bliver gemt i.
